## 3. Contribute Code

### Getting Started

1. **Setup**

   ```bash
   # Fork the repository on GitHub, then clone your fork
   git clone https://github.com/YOUR_USERNAME/eternal-motion.git
   cd eternal-motion
   npm install
   ```

2. **Development**

   ```bash
   npm run dev  # Start development server at http://localhost:3000
   npm run lint # Check code style
   npm test     # Run tests
   ```

3. **Submit Changes**

   ```bash
   # Create a new branch
   git checkout -b your-feature-name

   # Make your changes, then commit
   git add .
   git commit -m "Description of changes"

   # Push to your fork
   git push origin your-feature-name
   ```

4. **Open Pull Request**
   - Go to your fork: `https://github.com/YOUR_USERNAME/eternal-motion`
   - GitHub will show a banner "Your branch is ahead" with a "Compare & pull request" button
   - Click "Compare & pull request"
   - Fill in the description of your changes
   - Click "Create pull request" to submit your changes to the original repository
