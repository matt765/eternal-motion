# Eternal Motion

A collection of interactive physics-based animations and simulations built with React, Three.js, and Matter.js.

## Features

- **Spinner** - Dynamic spinning simulation
- **Solar System** - 3D orbital mechanics visualization
- **Cloth Swing** - Realistic cloth physics simulation
- **Slingshot** - Projectile motion with elastic forces
- **Atom Model** - Animated atomic structure visualization
- **Wrecking Ball** - Pendulum dynamics and collision physics
- **Newton's Cradle** - Conservation of momentum demonstration

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- [Next.js](https://nextjs.org/) - React framework
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) - React renderer for Three.js
- [Three.js](https://threejs.org/) - 3D graphics library
- [Matter.js](https://brm.io/matter-js/) - 2D physics engine
- [Framer Motion](https://www.framer.com/motion/) - Animation library
- [Zustand](https://zustand-demo.pmnd.rs/) - State management
- [Sass](https://sass-lang.com/) - CSS preprocessor
- [TypeScript](https://www.typescriptlang.org/) - Type safety

## Project Structure

